# 视觉打磨：像素风二期（盘面入屋 · 像素图标 · 氛围细节）

日期：2026-06-13
范围：在已合并的像素风一期（2026-06-12）之上做视觉打磨。`index.html` 为主（CSS + canvas 的 `drawPegboard` + 图标相关的少量 JS 文本切换行）；不动游戏逻辑、存档、server、协议。

## 背景

一期完成后整体统一，但复查发现 5 处可明显拔高的点（已与用户逐条确认，全做）：

1. **盘面底板**仍是大圆角软垫（canvas `drawPegboard` 画 `roundRect(..., s*0.5)`），是全屏最大元素与最大风格断层。
2. **emoji 功能图标**（💡🖌🧽🌙🔊❓🖼🏆😌🔥🧼↩︎🎲 等）是最后的非像素元素。
3. **木桌纹理**太平：只有拼缝线+渐变光。
4. **调色板**孤槽漂在羊皮纸上，缺「豆子盒」容器感。
5. **小氛围件**：夜间油灯光圈、CC 空状态像素图、header 木牌 tagline。

## 一、盘面底板像素化（canvas）

豆子与孔**继续不动**（作品）；只改底板容器绘制：

- `drawPegboard` 的大圆角 `roundRect(..., s*0.5)` 改为**方角矩形** + 像素木框：
  1. 先整块填深木描边色（新 CSS 变量 `--peg-stroke`，日 `#8a6a42` / 夜 `#1f1409`），矩形外扩量保持现有 `s*0.35`；
  2. 内缩描边宽 `bw = max(2, round(s*0.12))` 填 `TH.peg`（现有底板色不变）；
  3. 顶部画一条 `bw` 高的半透明白条（受光）、底部一条半透明黑条（背光）作 bevel；
  4. 孔点循环原样保留。
- `applyThemeVars()` 增读 `--peg-stroke`（带回退默认值，与现有 TH 字段同模式）。
- 接触阴影（豆子下的 `rgba(55,40,20,0.20)` roundRect）不属于底板，不动。

## 二、木桌纹理 + 夜间油灯光圈（纯 CSS）

- 新变量：`--plank-hi`（板面受光微提亮）、`--plank-lo`（板面背光微压暗）、`--grain`（细木纹）：
  - 日：`--plank-hi:rgba(255,245,220,.05)`、`--plank-lo:rgba(74,50,32,.05)`、`--grain:rgba(74,50,32,.045)`
  - 夜：`--plank-hi:rgba(255,220,160,.025)`、`--plank-lo:rgba(0,0,0,.10)`、`--grain:rgba(0,0,0,.10)`
- `body` 与 `#showcase` 背景在现有「光晕 + 拼缝 + desk」三层之间加两层：
  - 板间明暗交替：`repeating-linear-gradient(0deg, var(--plank-hi) 0 46px, var(--plank-lo) 46px 92px)`（与 46px 拼缝同周期，两板一循环）
  - 细竖木纹：`repeating-linear-gradient(90deg, transparent 0 7px, var(--grain) 7px 8px)`
- 夜间光圈更聚拢成「油灯」：`[data-theme="night"] body` 覆盖整条 background，把 radial 改为 `radial-gradient(700px 480px at 82% -6%, var(--desk-glow), transparent 55%)`，其余层同 body；夜 `--desk-glow` 提暖为 `#6b3f15`。`#showcase` 夜间同理覆盖。

## 三、像素图标系统（SVG sprite，替换所有功能 emoji）

- `<body>` 顶部内嵌一个隐藏的 `<svg>` sprite，若干 `<symbol viewBox="0 0 12 12">`，全部用**整数坐标的 `<rect>`** 拼 12×12 像素画（不用曲线），颜色用 `currentColor` 为主 + 少量固定色块。
- 使用处：`<svg class="pi"><use href="#pi-xxx"/></svg>`；CSS：`.pi{width:15px;height:15px;flex:none;display:inline-block;vertical-align:-2px}`（12 的非整倍缩放在 SVG rect 下仍锐利；悬停继承按钮文字色）。
- 图标清单（21 个 symbol，已对照 HTML 逐一确认替换处）：
  `pi-logo`(🧩 header logo + wdReplay「重新拼这张」) · `pi-picture`(🖼 btnPick) · `pi-trophy`(🏆 btnGallery + 作品墙标题) · `pi-relax`(😌 btnGuided 无忧态 + 帮助行) · `pi-palette`(🎨 btnGuided 自由态 + 帮助行) · `pi-bulb`(💡 btnHint) · `pi-brush`(🖌 btnMode) · `pi-eraser`(🧽 btnErase) · `pi-search`(🔍 btnCheck + 帮助行) · `pi-eye`(👁 btnLiveCheck) · `pi-moon`/`pi-sun`(🌙/☀️ btnTheme 两态) · `pi-sound`/`pi-mute`(🔊/🔇 btnSound 两态) · `pi-help`(❓ btnHelp) · `pi-undo`(↩︎ btnUndo) · `pi-iron`(🔥 btnIron) · `pi-clear`(🧼 btnClear) · `pi-dice`(🎲 btnRandom) · `pi-camera`(📷 上传区) · `pi-sleep`(CC 空状态装饰图，24×24)
- **JS 小改（仅 3 处图标切换行，逻辑不动）**：`index.html:583`（btnTheme 🌙↔☀️）、`index.html:1668`（btnSound 🔊↔🔇）、`index.html:1693`（btnGuided 😌无忧↔🎨自由）。改为重设按钮 innerHTML 为「`<svg class="pi"><use href="#pi-xxx"/></svg>` + 文字」。
- **帮助浮层里指代控件的 emoji**（😌/🎨/🔍 三个 help-row）同步换成对应 `.pi` 图标，与按钮视觉一致。
- **保留 emoji 的地方**：纯文案语气 emoji（空状态 ✨、toast 文案、`.swatch.done` 的 ✓、README）不换——它们是文字内容不是控件图标。
- **像素画产出方式**：21 个 symbol 的 rect 像素画由实现者绘制（设计稿不逐 rect 指定），合入前由 controller 在 15px 实渲染下逐个目检，不达标的重画——像素画是内容，验收以视觉为准。

## 四、豆子盒 + 木牌 + 空状态（CSS/HTML 小改）

- **调色板豆子盒**：`.palette` 从平羊皮纸改为**凹陷木槽内胆**：背景 `var(--soft)` 加一档木色 `linear-gradient(180deg, var(--chip), var(--soft))`，`box-shadow:inset 0 3px 0 var(--bevel-dark), inset 0 -2px 0 var(--bevel-light)`，上边框保持 3px。槽位（已是凹陷 slot）浮在槽里，空位自然延伸。
- **header 木牌 tagline**：`.sub`（“等代码的间隙，拼一颗豆”）做成挂在横梁上的小木牌：深木底（`color-mix(in srgb, var(--desk) 70%, #000)` 级别的暗木）、2px `--line` 边框、浅米字、左右上角两颗「钉子」用 `::before/::after` 4px 方点；不旋转（像素风不歪）。
- **CC 空状态像素图**：`.empty-hint` 顶部加一个 24×24 的像素「睡觉的显示器」SVG（sprite 里加 `pi-sleep` symbol，viewBox 0 0 24 24，整数 rect），下方文字不变；颜色用 `var(--muted)`。

## 五、不做清单（YAGNI）

- 不做 canvas 豆子/孔/接触阴影的任何改动；不做撒花/音效改动。
- 不做自定义鼠标指针、不做按钮按下音效、不做 CRT/扫描线滤镜（与「温柔解压」气质冲突）。
- 帮助浮层 kbd 内的字符、文案 emoji 不在图标替换范围。

## 六、验证

- `node --test` 全绿（图标 JS 小改不触碰被测逻辑，回归兜底）。
- 预览截图：日/夜 × 主界面（重点盘面木框与桌面纹理）、所有带图标的控件（hover 继承色）、主题/音量切换图标动态切换正确、调色板豆子盒、CC 空状态、header 木牌、作品墙。
- 图标可读性：15px 渲染下逐个目检；夜间对比度过一遍。
