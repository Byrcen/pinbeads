# 像素风三期收尾（最后一公里 · 7 小项）

日期：2026-06-13
范围：`index.html` 单文件，7 个微项，3 个 commit（CSS 包 / 视觉包 / 音效包），音效独立成 commit 便于按耳朵回退。逻辑、存档、server 不动。

来源：二期合并后的「还没做」盘点，用户拍板全做。

## Commit 1 — CSS 包（滚动条 / 复选框 / Firefox 滑杆）

1. **像素滚动条**（全局）：
   ```css
   ::-webkit-scrollbar{width:12px; height:12px}
   ::-webkit-scrollbar-track{background:var(--chip)}
   ::-webkit-scrollbar-thumb{background:var(--line); border:2px solid var(--chip)}
   ::-webkit-scrollbar-corner{background:var(--chip)}
   ```
   Firefox 回退：`html{scrollbar-color:var(--line) var(--chip)}`（细滚动条不强求方角）。
2. **筛选复选框**：`.fp-item input` 去原生外观——`appearance:none` + 14×14 方框（2px `--line` 边、`--panel-2` 底）；`:checked` 填充 `--accent` + 内 bevel 双 inset。原 `accent-color`/尺寸行替换。
3. **Firefox 滑杆把手**：补 `.toolbar input[type=range]::-moz-range-thumb`，属性同 webkit 版（16×22、panel-2 底、3px 边、bevel、radius 0）+ `border-radius:0`。

## Commit 2 — 视觉包（favicon / 标错方框 / 撒花步进）

4. **像素 favicon**：`<head>` 加 `<link rel="icon" href="data:image/svg+xml,...">`，内容为 pi-logo 同构的 12×12 豆粒环，粉色 `#ff5a7a`（URL 编码 `%23ff5a7a`），透明底。
5. **markWrong / markWrongLive 方角化**（canvas，仅这两个反馈标记函数）：
   - `markWrong`：圆角圈 `roundRect+stroke` → `strokeRect(px+s*0.08, py+s*0.08, s*0.84, s*0.84)`；右上圆点 `arc` → `fillRect(px+s*0.72, py+s*0.10, s*0.18, s*0.18)`。线宽/颜色不变。
   - `markWrongLive`：白底圆点+红圆点 → 白方块 `fillRect(px+s*0.68, py+s*0.08, s*0.26, s*0.26)` + 红方块 `fillRect(px+s*0.71, py+s*0.11, s*0.20, s*0.20)`。
6. **撒花粒子步进旋转**：`confetti()` 绘制处 `fctx.rotate(p.r)` → `fctx.rotate(Math.round(p.r/(Math.PI/4))*(Math.PI/4))`（45° 八向步进翻滚，物理不变）。

## Commit 3 — 音效包（克制的 8-bit 化，独立可回退）

7. 仅换振荡器波形与音量补偿（方波响度高，降量 30-40%），包络/时值/噪声层全部不动：
   - `place()`：`'triangle',0.16` → `'square',0.10`
   - `complete()`：`'sine',0.16` → `'square',0.11`
   - `ccAlert()`：两处 `'sine',0.28` → `'square',0.20`（保持最响，它是核心提醒）
   - `cool()`：两处 `'sine'` → `'square'`，音量 `0.12→0.09`、`0.16→0.12`
   - `peel()` 的尾音 tone：`'sine',0.2` → `'square',0.14`
   - `iron()/flip()` 是纯噪声，不动；`tone()` 函数本身与默认参不动。

## 不做

- 音效以外的一切听感设计（混响/bitcrusher 等）。
