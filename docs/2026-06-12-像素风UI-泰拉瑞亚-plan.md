# 像素风 UI（泰拉瑞亚 · 木质/羊皮纸）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把拼豆的 UI 框架（header/按钮/面板/调色板/操作台/弹窗/作品墙/toast）全量替换为泰拉瑞亚式木质/羊皮纸像素风，保留日/夜切换，盘面与豆子渲染不动。

**Architecture:** 方案 A「原地改造」——保留 `:root` / `[data-theme="night"]` CSS 变量管道，替换变量值并新增 `--bevel-light/--bevel-dark/--plank` 像素造型变量；逐组件把圆角/柔阴影/细边框改写为方角/硬阴影/3px 木框 bevel。新增 `assets/fonts/` 像素字体由 `server.js` 新静态路由 serve。JS（canvas 渲染、游戏逻辑）零改动。

**Tech Stack:** 纯 CSS（零图片素材）、Fusion Pixel 12px woff2（OFL-1.1）、Node 原生 http（无依赖）、node:test。

**对应 spec:** `docs/2026-06-12-像素风UI-泰拉瑞亚-design.md`

**通用约定（每个 CSS 任务都遵守）：**
- 引用行号以当前 `index.html`（commit `fd238f3`）为准，样式块在 7–330 行。先 Read 再 Edit，按内容匹配不要盲按行号。
- 像素阴影规则：**方角元素**用外部硬阴影 `var(--shadow)`；**切角元素**（`clip-path`）不带外阴影（clip 会把阴影裁掉），用 inset bevel 代替。
- 动效规则：删除 hover/配色类 `transition`（状态切换「啪」一下）；保留出现/位移类动画（banner 滑入、toast 滑入、progress 宽度），呼吸类动画改 `steps()` 闪烁。
- 每个任务完成后用预览实例目视核对涉及组件（日间+夜间各看一次），再 commit。

```js
'use strict';
const { test } = require('node:test');
const assert = require('node:assert');
const { spawn } = require('node:child_process');
const path = require('path');

// 以子进程方式拉起 server（PINBEADS_PORT=0 随机端口），从启动日志解析实际端口
function startServer() {
  return new Promise((resolve, reject) => {
    const child = spawn(process.execPath, [path.join(__dirname, '..', 'server.js')], {
      env: { ...process.env, PINBEADS_PORT: '0' },
    });
    let out = '';
    child.stdout.on('data', (c) => {
      out += c;
      const m = out.match(/localhost:(\d+)/);
      if (m && Number(m[1]) > 0) resolve({ child, port: Number(m[1]) });
    });
    child.on('error', reject);
    setTimeout(() => reject(new Error('server 未就绪，stdout: ' + out)), 5000).unref();
  });
}

test('GET /assets 字体：200 + font/woff2 + wOF2 魔数', async () => {
  const { child, port } = await startServer();
  try {
    const r = await fetch(`http://localhost:${port}/assets/fonts/fusion-pixel-12px-proportional-zh_hans.woff2`);
    assert.equal(r.status, 200);
    assert.equal(r.headers.get('content-type'), 'font/woff2');
    const buf = Buffer.from(await r.arrayBuffer());
    assert.equal(buf.subarray(0, 4).toString('ascii'), 'wOF2');
  } finally { child.kill(); }
});

test('路径穿越（/assets/..%2Fserver.js）返回 404', async () => {
  const { child, port } = await startServer();
  try {
    const r = await fetch(`http://localhost:${port}/assets/..%2Fserver.js`);
    assert.equal(r.status, 404);
  } finally { child.kill(); }
});

test('不存在的资源返回 404', async () => {
  const { child, port } = await startServer();
  try {
    const r = await fetch(`http://localhost:${port}/assets/fonts/nope.woff2`);
    assert.equal(r.status, 404);
  } finally { child.kill(); }
});
```
